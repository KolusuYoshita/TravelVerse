TravelVerse is a visually stunning, frontend-only travel guide platform designed for modern travelers. It provides an immersive experience for exploring global destinations, filtering through travel categories, and simulating the booking process with a focus on high-quality UI/UX.

Key Features:

- Interactive Destination Discovery: Users can explore various travel spots with high-quality imagery and detailed descriptions.
- Destination Filtering: Dynamic filtering system to sort travel locations based on user preferences.
- Booking Simulation: Includes interactive booking confirmation modals to mimic a real-world reservation flow.
- Mobile-First Design: Fully responsive layouts that adapt perfectly to smartphones, tablets, and desktops.
- Cross-Browser Compatibility: Optimized to ensure a consistent experience across all modern browsers.

Tech Stack:

- HTML5: For structured and semantic web content.
- CSS3: For advanced styling, flexbox/grid layouts, and smooth animations.
- JavaScript: For DOM manipulation, destination filtering logic, and interactive modals.

Open the index.html file in any web browser to view the site.
